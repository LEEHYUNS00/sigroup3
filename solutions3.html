Zetworks-WMC

(웹메일 클라이언트)

검색엔진 기반 웹메일 시스템

  • 대용량 검색 지원을 위한 NoSQL(Elasticsearch) 기반의 웹메일 클라이언트 시스템
  • Zetworks-WMC 구축, SMTP, POP3 등 메일 중계 서버 별도 운영 필요

Main Function

  • Collect mail
    • 메일수집
    • 공용 메일을 위한 별도의 사용자 메일 할당 기능
  • Convenience
    • 검색엔진 기반 메일 수집을 통한 빠른 검색 속도(사용자, 제목, 본문, 첨부파일명, 범주 등)
    • 다중 필터 기능(적용규칙 : N, 예외규칙 : N, 처리규칙 : N)
    • 다양한 처리 규칙 제공(폴더이동, 범주할당, 읽음처리, 삭제처리, 특정메일로 포워딩)
    • 범주 관리 기능
    • 발송계정 설정을 통한 다수의 공용 메일 계정 사용자를 위한 발신자 설정 기능
    • 연락처 자동 수집 기능 및 분류
    • 멀티 서명 관리를 통한 다수의 메일 계정 관리 편의
    • 신규 메일 작성 시 기존 메일 첨부 및 기존 메일내 첨부파일 검색 첨부 기능
    • 특정 조건에 따른 사용자+메일계정 할당 기능(특정 기간 이후, 특정 필터 조건 등)
    • 서버 기반의 사용자 운용 환경 설정
    • EML 파일 관리자 복원 기능

Sample Creation Cases

  • Case
    • 공용 계정을 통한 전세계 선사 견적 및 발주 관리
    • 기존 아웃룩 메일 클라이언트를 통한 사용자 환경
      • → 메일함 용량 및 공용 계정에 대한 다수의 사용자 처리로 인한 부하, 보안, 운영 문제점 존재
    • Zetworks-WMC를 통한 공용계정 관리, 검색, 필터 등 개별 환경으로 전환
    • 대량의 메일처리 시 병목 발생 현상 해결, 관리자에 의한 신규 사원입사시 공용 계정 메일 전달 범위 지정
    • 일반적인 메일 처리는 기존 웹메일 클라이언트 솔루션과 동일
    • 검색엔진 기반 메일 저장소를 통한 신속한 검색 결과 및 필터 정보 제공
